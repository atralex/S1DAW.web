Párrafos

Los parrafos se añaden con el comando p, para cambiar de linea se puede utilizar otro comando p o se puede cerrar el parrafo y abrir otro.

Ahora hemos cambiado de parrafo, de esta forma deja una linea en blanco entre medias, sin embargo, si utilizamos br.
Nos cambia de linea sin dejar una en blanco.

En cuanto al formato de letra, si utlilizamos el comando strong o em utilizamos la negrita y la cursiva respectivamente.

Para poner encabezados con este tamaño, utilizamos el comando h1

Para poner encabezados con este tamaño, utilizamos el comando h2

Para poner encabezados con este tamaño, utilizamos el comando h3

Para poner encabezados con este tamaño, utilizamos el comando h4, y asi seguimos haciendo

Otras veces se necesita hacer un sub o un superindice, con la etiqueta sub o sup.

Y en caso de necesitar una abreviatura utilizamos la etiqueta abbr CaSPA
